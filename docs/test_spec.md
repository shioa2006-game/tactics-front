# 自動テスト仕様書

## 概要

ゲームバランス調整を目的とした自動対戦システム。異なる戦略を持つAI同士を対戦させ、ユニットバランス・マップバランス・ゲームシステムの健全性を数値的に検証する。

---

## テストの目的

- ユニットバランス: 各ユニットの使用頻度・勝率・価値の適正性
- 戦略多様性: 特定戦略の支配的優位性がないかの確認
- マップバランス: 地形・陣地配置の公平性検証
- ゲーム展開: 適切なゲーム時間・逆転可能性の評価

---

## バランス評価指標

```
ユニットバランス
- ユニット勝率: 各ユニットが関与した勝利 45〜55% が理想
- ユニット使用率: 各戦略で最低30%以上
- コスト効率: ユニット間のコスト効率が同程度

戦略バランス
- 戦略間勝率差: 各対戦で55〜65%程度の差
- ゲーム長: 15〜30分が適正範囲
- 逆転率: 中盤劣勢から逆転率20%以上

システムバランス
- 陣地価値: 陣地間の価値均一性
- 先手後手の勝率差: 10%以内
```

## AI仕様

### Phase 1で実装するAI

**RandomAI**
- アルゴリズム: 完全ランダム行動
- 用途: 動作確認・最低ベースラインの確認

**GreedyAI**
- アルゴリズム: 1手先読み（最も近い目標への移動）
- 用途: 基本的な対戦相手・RandomAIとの差異確認

### Phase 2で実装するAI

**MinMaxAI（深度2）**
- アルゴリズム: MinMax + Alpha-Beta枝刈り
- 評価関数: 陣地占拠・ユニット価値・戦術的優位性の総合評価
- 目標思考時間: 2〜5秒

**AggressiveAI（攻撃重視型）**
```
基本戦略: 敵ユニット撃破を最優先
評価重み:
- 陣地占拠: 5点
- 敵ユニット撃破: 30点
- 自軍ユニット生存: 10点
- 敵への接近ペナルティ: -3点

特徴:
- 序盤: 敵に向かって一直線
- 中盤: 弱いユニットを集中攻撃
- 終盤: 陣地争いで不利になりがち
```

**DefensiveAI（守備重視型）**
```
基本戦略: 陣地確保と安全な範囲での攻撃
評価重み:
- 陣地占拠: 20点
- 自軍ユニット生存: 25点
- 敵ユニット撃破: 10点
- 陣地安全度: 15点

特徴:
- 序盤: 安全な陣地を確実に占拠
- 中盤: 防御ライン構築
- 終盤: 消極的で逆転を許すリスクあり
```

**TerritorialAI（陣地重視型）**
```
基本戦略: 効率的な陣地占拠でポイント稼ぎ
評価重み:
- 陣地占拠: 30点
- 陣地占拠効率: 20点
- ユニット生存: 15点
- 戦闘: 5点
- 陣地間移動効率: 10点

特徴:
- 序盤: 複数陣地への同時攻略
- 中盤: 陣地争奪戦での戦術的優位
- 終盤: 陣地数優位で安定リード
```

### Phase 3で強化するAI

**MinMaxAI（深度3）**
- 深度2からの拡張
- 手数フィルタリング（上位10〜15手のみ評価）
- 目標思考時間: 1〜3秒

## バッチ実行仕様

### Phase 1（ブラウザ上、メインスレッド）

- 実行件数: 100戦（動作確認用）
- 対戦組み合わせ: RandomAI vs GreedyAI
- 実行方法: test.html をブラウザで開いて実行ボタンを押す
- 出力: 勝率・平均ターン数・ゲーム完走率をtest.html上に表示

### Phase 2（ブラウザ上、メインスレッド）

- 実行件数: 1000戦
- 対戦組み合わせ:
  - MinMaxAI vs AggressiveAI
  - MinMaxAI vs DefensiveAI
  - MinMaxAI vs TerritorialAI
  - AggressiveAI vs DefensiveAI
  - AggressiveAI vs TerritorialAI
  - DefensiveAI vs TerritorialAI
- 出力: 勝率・ユニット使用率・戦略別分析をtest.html上に表示

### Phase 3（Web Worker使用）

- 実行件数: 10,000戦
- 実行方法: Web Workerで非同期実行（画面フリーズなし）
- 進捗表示: リアルタイムで進捗率をtest.html上に表示
- 目標完了時間: 30分以内
- 出力: 全指標の可視化レポート（グラフ・ヒートマップ・時系列）

## test.htmlの画面仕様

### Phase 1時点の表示内容

- 実行ボタン（バッチ開始）
- 進捗表示（完了件数 / 総件数）
- 結果サマリー
  - 総対戦数
  - 各AIの勝率
  - 平均ターン数
  - ゲーム完走率

### Phase 2で追加する表示

- 対戦組み合わせ別の勝率一覧
- ユニット別使用率・生存率・撃破数
- 戦略別の特徴分析テキスト
- バランス判定（評価指標との比較）

### Phase 3で追加する表示

- 進捗バー（リアルタイム更新）
- 勝率推移グラフ
- ユニット相性マトリクス
- マップヒートマップ（陣地占拠頻度）
- 自動バランス提案テキスト
