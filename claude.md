# CLAUDE.md

このファイルはClaudeCodeが常に参照するプロジェクト全体の制約・構造定義です。
コーディング開始前に必ず読み、全ての指示に従ってください。

---

## プロジェクト概要

32×32マップ上で行うターン制戦略ゲーム。プレイヤーとCPUがユニットを使って陣地を争奪し、ターン終了時のポイントで勝敗を決める。ゲームバランス調整を目的とした自動対戦テストシステムを並行して開発する。

---

## 技術スタック

- 言語: JavaScript（ピュアJS、TypeScriptは使用しない）
- 描画: Canvas API（p5.jsなどの描画ライブラリは使用禁止）
- 実行環境: ブラウザ（サーバー不要、静的ファイルのみ）
- バッチテスト: Web Worker（Phase 3から導入）
- バージョン管理: Git / GitHub

## 設計の絶対ルール

以下のルールはいかなる場合も破ってはいけません。

**ロジックと描画の分離**
- src/game/ 配下のファイルはCanvas APIやDOMに一切触れない
- src/renderer/ 配下のファイルはゲームの状態を変更しない
- この分離により、Web WorkerでゲームロジックのみをHeadlessで実行できる

**ピュアJSの維持**
- p5.js、Three.jsなどの描画ライブラリは使用しない
- 外部ライブラリは原則使用しない
- Node.js固有のAPIは使用しない（将来的なブラウザ動作を保証するため）

**ファイル分割の方針**
- 1ファイル1クラスを基本とする
- 1ファイルが300行を超える場合は分割を検討する
